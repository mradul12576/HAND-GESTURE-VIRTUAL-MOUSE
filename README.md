# HAND-GESTURE-VIRTUAL-MOUSE
**🖐️ Hand Gesture Virtual Mouse** lets you control your computer using hand gestures captured by a webcam 🎥. Built with **Python, OpenCV, MediaPipe, and PyAutoGUI**, it tracks hand landmarks and converts finger movements into cursor actions like move 🖱️, click 👆, and scroll 🔄, enabling smooth and touchless computer interaction 🚀.
## 🚀 Features

- Real-time hand tracking using MediaPipe
- Cursor movement using index finger
- Left click and right click gestures
- Scroll using two-finger movement
- Smooth cursor control
- Touchless computer interaction

---

## 🧠 How It Works

1. The webcam captures live video frames.
2. OpenCV processes the frames.
3. MediaPipe detects hand landmarks and finger positions.
4. The system identifies gestures based on finger combinations.
5. PyAutoGUI converts gestures into mouse actions.

---

## 🖐️ Gesture Guide

| Gesture | Action |
|-------|-------|
| Index Finger Up 👆 | Move Cursor |
| Index + Middle Finger Up ✌️ | Enable Tracking |
| Thumb + Index Pinch 🤏 | Left Click |
| Thumb + Middle Pinch | Right Click |
| Two Fingers Moving 🔄 | Scroll |
