# HAND-GESTURE-VIRTUAL-MOUSE
**🖐️ Hand Gesture Virtual Mouse** lets you control your computer using hand gestures captured by a webcam 🎥. Built with **Python, OpenCV, MediaPipe, and PyAutoGUI**, it tracks hand landmarks and converts finger movements into cursor actions like move 🖱️, click 👆, and scroll 🔄, enabling smooth and touchless computer interaction 🚀.
